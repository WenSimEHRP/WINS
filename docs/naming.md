### Prefixes

| Prefix | Meaning                                                                             |
|--------|-------------------------------------------------------------------------------------|
| id     | Index to a subject. The subject can be a spriteset, a spritegroup, or anything else |
| param  | a parameter, or a value from an external parameter                                  |
| s      | a spriteset                                                                         |
| sp     | a spritelayout                                                                      |
| sw     | a switch                                                                            |
| i      | an item                                                                             |
| tmpl   | a template                                                                          |
| t      | an index to a temp register                                                         |
